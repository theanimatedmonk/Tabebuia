# Tabebuia

A React project with [Rive](https://rive.app) animation integration, built with Vite.

## Getting Started

```bash
npm install
npm run dev
```

## Stack

- React 19
- Vite 7
- `@rive-app/react-webgl2` (Rive Renderer)

See [RIVE_SETUP_AND_IMPLEMENTATION.md](./RIVE_SETUP_AND_IMPLEMENTATION.md) for a comprehensive guide on Rive + React setup, data binding, view models, and advanced features.
